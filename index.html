Bandwidth Humana

Quão eficiente seria uma conexão utilizando humanos como transmissores de informação? Se a tecnologia de 4G pudesse ser substituída por um humano, quão (in)eficaz seria?

Humano padrão

Escrita ("upload")

Média humana de 32.5 WPM (palavras por minuto).

Cada palavra tem, em média na língua inglesa, 4.79 caracteres. Portanto, a escrita tem uma velocidade média de 32.5 × 4.79 = 155.675 CPM.

Leitura ("download")

Média humana de 863 CPM. (caracteres por minuto)

Por simplicidade, utilizando como codificação ASCII, podemos assumir que cada caractere requer 7 bits para ser representado.

155.675 × 7 = 1089.725 bits/minuto.

1089.725 / 60 = 18.162 bits/segundo.

18.162 / 1000 = 0.018 Kbps.

863 × 7 = 6041 bits/minuto.

6041 / 60 = 100.683 bits/segundo.

100.683 / 1000 = 0.101 Kbps.

Para efeito de comparação, uma internet móvel 2.5G (GPRS) tem uma velocidade teórica de 14 Kbps para upload e 40 Kbps para download.

Isso é uma velocidade de upload:

14 / 0.018 = 777.78 vezes maior.

Isso é uma velocidade de download:

40 / 0.101 = 396.04 vezes maior.
